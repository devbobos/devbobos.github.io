### Performance Monitoring 기능 업데이트
- 2017년 기준 Beta로 제공되는 기능이다
- 상세 설명
  - Traces 포인트를 지정하여 시간 별로 데이터를 수집한다
  - Network Requests 네트워크 트래킹 기능 지원
  - Tracking 항목
    - App version
    - Device
    - Country
    - OS Level
    - Carrier
    - Radio
  - Test Lab : 다양한 화면 및 기종 테스트를 클라우드로 지원한다!
  - Google Analytics for Firebase
